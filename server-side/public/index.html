API Documentation

Welcome to the API Documentation of video Hosting WebApp!

Available Routes:

  • User routes

  • POST /api/v1/users/register - Create a new user.
  • POST /api/v1/users/login - User can log in.
  • GET /api/v1/users/find/:id - find a user by id.
  • POST /api/v1/users/refresh-Token - User can refresh the access token .

    • Protected routes

  • POST /api/v1/users/logout - User can log out.
  • POST /api/v1/users/change-password - Update an existing user password.
  • PATCH /api/v1/users/update-account - Update an existing user.
  • DELETE /api/v1/users/delete-user - Delete a user
  • PUT /api/v1/users/sub/:id - current user can subscribe a video by videoId
  • PUT /api/v1/users/unsub/:id - current user can unsubscribe a video by videoId
  • PUT /api/v1/users/like - current user can like a video by videoId
  • PUT /api/v1/users/dislike - current user can dislike a video by videoId
  • PUT /api/v1/users/save -current user can save a video by videoId

  • Video routes

  • GET /api/v1/videos/find/:id - can get a video by videoId
  • PUT /api/v1/videos/add-view - no. of users watches a video
  • GET /api/v1/videos/random - random videos
  • GET /api/v1/videos/trend - trending videos
  • GET /api/v1/videos/search - video can be search by title

    • Protected routes

  • POST /api/v1/videos - current user can upload a video
  • PUT /api/v1/videos/:id - current user can update a video by videoId
  • DELETE /api/v1/videos/:id - current user can delete a video by videoId
  • GET /api/v1/videos/tags - get a video by tags
  • GET /api/v1/videos/sub - subscribed videos

  • Comment routes

  • POST /api/v1/comments - current user can add a comment on video
  • DELETE /api/v1/comments/:id - current user can delete a comment
  • GET /api/v1/comments/:videoId - get all the comments on video
